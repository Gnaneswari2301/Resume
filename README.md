# Pakki Gnaneswari - Portfolio

A modern, responsive portfolio website showcasing my work as a Full Stack Developer and Cloud Computing enthusiast. Built with React, Vite, and Tailwind CSS.

## Features

- Responsive design with dark theme
- Animated components using Framer Motion
- Project showcase with case studies
- Skills and certifications display
- Contact form and social links
- Downloadable resume

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start development server
   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```

## Deployment

This project is configured for static site deployment. The built files in `dist/` can be deployed to:

- GitHub Pages
- Vercel
- Netlify
- Any static hosting service

### GitHub Pages Deployment

1. Build the project: `npm run build`
2. Commit and push the `dist/` folder to your GitHub repository
3. Enable GitHub Pages in repository settings, selecting the `dist/` folder as source
